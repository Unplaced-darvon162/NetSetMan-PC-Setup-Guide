## 📋 What is NetSetMan?

NetSetMan acts as a network setting manager for your computer. It switches between network configurations with one click. You save time when you move between home, work, and public networks. It handles IP addresses, WiFi connections, and printer defaults.

## ⚙️ Handling Setup Errors

Windows 11 often blocks software to protect your system. You might see a message stating that the app is blocked or that Windows protected your PC. Follow these steps to bypass these blocks.

### Unblocking the Installer

If a window appears saying "Windows protected your PC," perform these actions:

1. Click the "More info" link in the blue text.
2. Click the "Run anyway" button that appears.
3. Allow the installer to make changes to your device.

### Administrative Permissions

The software requires full access to your network hardware. You must run the installation as an administrator. Right-click the installer icon and select "Run as administrator" from the menu.

### Checking Firewall Settings

Sometimes your firewall stops network tools. Open your Windows settings, go to Privacy and Security, then select Windows Security. Choose Firewall and network protection. Ensure the app has permission to communicate through the firewall.

## 🔧 Managing Your Profiles

Once you install the tool, you define network profiles. Each profile contains your unique settings for a specific location.

1. Launch the NetSetMan interface.
2. Assign a name to your first profile, such as "Home" or "Office."
3. Select your network adapter from the dropdown list.
4. Enter the IP address, Subnet Mask, Gateway, and DNS servers.
5. Click "Activate" to apply these settings to your hardware.

## 🛡️ Privacy and Performance

The application does not track your browsing history. It only interacts with your system network configuration files. It uses low system resources and remains quiet in the system tray when not in use. You can set the app to start with Windows if you need immediate connection management upon logging in.

## 🔍 Troubleshooting Common Issues

If you cannot connect to the internet after an update, disable and re-enable your network card. Open Device Manager, find Network Adapters, and toggle your card off and on. This forces Windows to refresh the adapter state after the software applies a new profile.
